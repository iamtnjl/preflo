# 20 — Organic Growth Strategy

> PRD Section: 8 | Priority: Phase 2 | Location: `apps/web`

## Purpose

Implement the organic growth strategy targeting "uncertainty queries" — the primary user acquisition channel.

## Tasks

### 20.1 Search Query Targeting
- [ ] Build pages targeting these query patterns per niche:

| Pattern | Example | Volume | Intent |
|---------|---------|--------|--------|
| "is [niche] worth it" | is blogging worth it | 500-5,000 | Pre-decision validation |
| "how hard is [niche]" | how hard is SEO | 200-2,000 | Difficulty assessment |
| "can I succeed in [niche]" | can I make money blogging | 100-1,000 | Feasibility check |
| "is [niche] saturated" | is AI content saturated | 200-1,500 | Market assessment |
| "future of [niche]" | future of SEO | 500-3,000 | Trend assessment |
| "[niche a] vs [niche b]" | blogging vs YouTube | 100-500 | Comparison decision |

### 20.2 Growth Loop Implementation
- [ ] Programmatic landing pages rank in search
- [ ] Users interact with charts and sliders (high session duration)
- [ ] Shareable result cards for social media
- [ ] Backlinks from shares improve domain authority
- [ ] More pages rank → more users → more outcome data → better accuracy → more trust

### 20.3 Viral Mechanics
- [ ] Shareable result cards (Twitter/LinkedIn optimized images)
- [ ] Embeddable widgets for agencies/bloggers
- [ ] Comparison challenges ("Which niche is harder?")
- [ ] Public API for data journalists

### 20.4 Content Strategy
- [ ] Monthly "State of the Internet" reports
- [ ] Methodology deep-dives (trust through transparency)
- [ ] Niche spotlights for trending topics
- [ ] Accuracy reports (calibration transparency)

### 20.5 SEO Implementation
- [ ] XML sitemaps (auto-generated)
- [ ] Structured data (FAQ, HowTo schemas)
- [ ] Internal linking structure
- [ ] Meta tags optimization
- [ ] OG images for social sharing
